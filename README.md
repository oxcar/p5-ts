# p5-ts

A p5.js and Typescript starter template

Clone repository

```shell
git clone https://github.com/oxcarh/p5-ts
```

Install npm dependencies
```shell
npm install
```

Install Parcel
```shell
npm install -g parcel-bundler
```

Development mode
```shell
npm run dev
```

Build for deployment
```shell
npm run build
```
